версия java: 22
запуск: java -jar SHIFT_task-1.0.jar in1.txt in2.txt
in1.txt и in2.txt, вместо этого имена ваших входных файлов или их абсолютный путь, если они находятся не в той же директории
возможные опции:
  -o задаёт новый путь для выходных файлов
  -p даёт префикс для имени выходных файлов
  -a включает режим добавления в файл
  -s выводит краткую статистику в терминал 
  -f выводит полную статистику в терминал
